# FroggyOS - 3D File Explorer

A Three.js-based underwater file browser where you control
a frog swimming through your files and folders.

## Features

- Third-person frog swimming controls (WASD + mouse)
- Realistic underwater environment with caustics
- File and folder visualization as 3D objects
- Multi-frog mode for collaborative browsing
- Dynamic lighting with real-time controls

## Controls

| Key | Action |
|-----|--------|
| W/A/S/D | Swim forward/left/back/right |
| Space | Dash boost |
| Shift | Swim up |
| Ctrl | Swim down |
| M | Toggle multi-frog mode |
| L | Lighting control panel |
| Enter | Preview selected file |

## Tech Stack

- Three.js for 3D rendering
- Custom GLSL shaders for bubble effects
- Vanilla JavaScript (no framework)

## Getting Started

1. Clone the repository
2. Run `npm install`
3. Start server: `npm run dev`
4. Open http://localhost:8000/swimming-tank.html
